Purpose

Currently the process for automatically transferring grades between Canvas for HAC is complicated, so many teachers have to manually enter grades in. To help simplify the process the project aims to use the scripting language within Excel (Visual Basic for Applications).

PreCondition/PostCondition for Application

Teachers can export their Canvas gradebook to a csv file (shown in the image above). The SIS User IDs in the Canvas csv file is equivalent to FISD student IDs. HAC accepts a csv file with a column containing student IDs and their grades for the chosen assignment. The Excel template should allow teachers to insert their Canvas csv file, enter the name of the assigment they want to transfer, and return a csv file for teachers to upload into HAC. Also, the Canvas csv file either returns a raw score (# of questions correct) or a percentage grade. The program will have to use conditional logic to test for this and calculate a percentage grade. The reason excel was chosen for this task is: 1) many teachers are familiar with the platform; 2) since the application deals with sensitive data, having it in excel promotes transparency and application security.
